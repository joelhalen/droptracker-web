# Creating a group

A group connects a **Wise Old Man group** with a **Discord server** so your clan
can track loot together.

## Before you start

- The **DropTracker bot** must be in your Discord server.
- You need **Manage Server** permission in that Discord server.
- Have your **Wise Old Man group ID** handy.

## The wizard

Go to [Create a group](/groups/new) and follow the three steps:

1. **WOM group** — enter your Wise Old Man group ID. We preview the group and
   confirm it isn't already registered.
2. **Discord server** — enter your server (guild) ID. We check the bot is present
   and that the server doesn't already own a group.
3. **Confirm** — set the group name and an optional Discord invite, then create.

You become the group **owner** and land on the admin dashboard, where you can
configure [notifications](/docs/group-settings), sync members, and more.
