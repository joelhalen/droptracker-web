# Lootboards

A **lootboard** shows a group's loot as a grid of item tiles, rendered live from
the latest data. Open it from any group page via the **Lootboard** button, or at
`/groups/{id}/lootboard`.

## Reading the board

- Tiles are **colored by value** — the rarer the drop, the richer the color.
- **Hover** a tile to see the item name, quantity, and total value.
- Use the **period switcher** to view all-time, monthly, weekly, or daily loot.

## Sharing

Prefer an image? The **Download image** button generates a shareable PNG of the
board — handy for posting to Discord or social media.
